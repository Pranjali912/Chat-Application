A Full Stack MERN Application.
 Tech stack: MERN , Socket.io , TailwindCSS , Daisy UI.
 Authentication && Authorization with JWT.
 Real-time messaging with Socket.io.
### Setup .env file

```js
PORT=...
MONGO_DB_URI=...
JWT_SECRET=...
NODE_ENV=...
```

### Build the app

```shell
npm run build
```

### Start the app

```shell
npm start
```
